# EventBook REST API

REST API application for managing events and booking places at events. Developed in Python using FastAPI.

## About the project

EventBook is an API for:
- User registration and authentication
- Event creation and management
- Event seat booking
- User info about booking and event

# Quick start

### Requirements

- **Python** 3.12+

## Installing dependencies

### Clone the repository
```
git clone <URL_REPOSITORY>
cd stas-corp-event_book
```

### Create and activate environment
```
python -m venv venv
venv\Scripts\activate
```

### Install dependencies
```
pip install -r requirements.txt
```

# Start the server

### Start on localhost:8000
```
python main.py
```

The server will be available at: http://localhost:8000

Swagger UI (Interactive Docs): http://localhost:8000/docs
